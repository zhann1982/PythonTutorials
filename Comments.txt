          Here are main codes for GIT:
----------------------------------------------------------------------------------------------------------------------------------------------

   git clone {repository link from github.com}   -> creates a folder on your computer with a link to repository

   git status   -> check if there are changes in current project

   git pull   -> to load latest changes from github

   git add fileName.extension    -> add editted file to the list of changes

   or

   git add -A     -> add all editted files to the list of changes

   git commit -m "comment on changes"    ->  commit changes before sending, add comment (comments will help others to see what has changed)

   git push    -> finally publish changes on Github
